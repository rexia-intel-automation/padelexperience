Wireframe lo-fi — site institucional Padel Experience

Somente estrutura e comportamento, sem identidade visual do cliente (tons de cinza, sem logo real). Todo conteúdo marcado com a tag CMS é editável pelo painel administrativo — o restante é fixo no template. O agendamento de quadras acontece sempre no app white-label Matchpoint; o site apenas direciona para as lojas. Use a navegação lateral para pular entre as 12 telas.
